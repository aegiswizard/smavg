# Smavg Assets

This folder contains the public visual assets used by the Smavg repo.

- `smavg-banner.svg`: original GitHub README banner.
- `smavg-mark.svg`: original Smavg project mark.
- `smavg-flow.svg`: original "repetition to exact expansion" flow diagram.
- `smavg-trust.svg`: original trust proof diagram.
- `smavg-dragon-temp.jpg`: temporary Smavg dragon mascot image owned by Roman
  Mataru and used with permission for this project.

The SVG files are original project assets. They do not copy Phoenix or any
third-party artwork.
