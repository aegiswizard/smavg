# Smavg 🐲 Roadmap

The roadmap is trust-first.

## Now

- CLI
- `.smavg` archive format
- context briefs
- exact expansion
- receipts
- ledger
- work mode
- skill
- MCP server
- plugin bundle
- conservative daemon
- 100-test trust suite

## Next

- polish install path
- improve public examples
- add more data-family codecs
- improve log/JSON/CSV cross-file sharing
- strengthen docs for ordinary users
- package releases cleanly

## Later

- Rust hot path after Python behavior is proven
- native binaries
- deeper daemon scheduling
- stronger incremental archive support
- filesystem layer after the core is trusted

## Rule

```text
Filesystem comes after trust.
Rust comes after behavior is proven.
No fake demos.
No shortcuts around exact restore.
```
