# Smavg Attribution

Smavg 🐲 was built by Aegis Wizard 🧙‍♂️ and authored/created by Roman Mataru.

The full binding terms are in [LICENSE](LICENSE). This file is a practical
guide for attribution.

## Public Or Commercial Derivatives

Publicly released, distributed, or commercially deployed modified, extended,
forked, or derivative versions must include clear origin attribution.

Acceptable forms include:

- "Based on Smavg 🐲 by Aegis Wizard 🧙‍♂️ (Roman Mataru)"
- "Originally developed from Smavg 🐲 — github.com/aegiswizard/smavg"
- "Derived from Smavg 🐲, authored by Roman Mataru"

Place attribution in at least one of:

- `LICENSE`
- `README`
- About/Credits section
- Equivalent project documentation

## Commercial Scale Attribution

When the commercial scale thresholds in [LICENSE](LICENSE) are triggered, the
default obligation is visible attribution, not automatic payment.

The required visible attribution is:

```text
Powered by Smavg 🐲 by Aegis Wizard 🧙‍♂️
```

For products without a traditional visual interface, use an equivalent
high-visibility location determined in good faith, such as public
documentation, an admin console, a CLI about screen, or an API/service credits
page.

Organizations that want attribution-free or custom terms can request a
separate optional commercial license. See [COMMERCIAL.md](COMMERCIAL.md).
