# Versioned Code History Example

This small folder shows the shape of versioned code.

It is not a benchmark. It is only for learning.

Try:

```bash
smavg pack examples/versioned-code-history --out /tmp/versioned-code.smavg
smavg verify /tmp/versioned-code.smavg
smavg restore /tmp/versioned-code.smavg /tmp/versioned-code-restored
```
