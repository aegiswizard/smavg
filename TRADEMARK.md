# Smavg Trademark And Brand Use

This file explains how the Smavg name and brand should be used.

The software license covers code and project materials.

The Smavg name, Smavg 🐲 identity, Aegis Wizard 🧙‍♂️ identity, visual marks,
mascot, and project branding are separate origin and brand identifiers.

## Allowed Uses

You may use Smavg names for truthful attribution, such as:

- "Based on Smavg 🐲 by Aegis Wizard 🧙‍♂️"
- "Derived from Smavg 🐲, authored by Roman Mataru"
- "Compatible with Smavg"
- "Uses Smavg under the Smavg Modified MIT License, Version Genesis 1.0 2026"

You may also use the required commercial-scale attribution when the license
thresholds are triggered:

```text
Powered by Smavg 🐲 by Aegis Wizard 🧙‍♂️
```

## Not Allowed

Do not:

- claim your fork is the official Smavg project
- imply endorsement by Roman Mataru, Aegis Wizard, or Smavg
- use Smavg branding in a confusing or misleading way
- remove origin attribution required by the license
- present modified versions as unmodified official releases

## Fork Naming

Forks should use a distinct name unless they are clearly identified as forks.

Good examples:

```text
Acme Archive, based on Smavg
Smavg-compatible compressor by Acme
```

Avoid:

```text
Official Smavg Pro
Smavg Enterprise by Acme
```

unless a separate written agreement allows it.
